# 기본 작업

vscode에서 index.html 기본 설정법 **!** 를 입력해서 아래와 같은 기본 세팅이 된다.
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```
